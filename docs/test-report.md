# 基于填图对象智能识别的文本空间化算法模块：测试报告（v1.0）

测试日期：2026-07-20

## 自动化验证

| 子系统 | 命令 | 结果 |
| --- | --- | --- |
| Vue 前端 | `pnpm build` | TypeScript 检查与 Vite 生产构建通过 |
| Spring Boot 后端 | `mvn clean test` | 11 项单元/编排测试全部通过 |
| FastAPI AI 服务 | `pytest` | 18 项文档、OCR 回退、实体、关系、空间、图谱、Qdrant、RAG 契约测试全部通过 |

覆盖重点：

- 文件格式和存储路径安全校验
- 文档解析、LLM 实体/关系/空间请求契约与异步编排
- GeoJSON 坐标范围、线面结构和闭环校验
- Nominatim 缓存与请求节流
- Neo4j 关系类型白名单与 Qdrant 索引/检索协议
- RAG 回答关联实体、空间位置和来源段落
- Excel 四工作表与 GeoJSON FeatureCollection 导出

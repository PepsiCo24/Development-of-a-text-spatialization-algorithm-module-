# Changelog

所有重要变更均记录在此文件中。

## [1.0.0] - 2026-07-20

### 新增

- Phase 8 Excel、CSV、JSON、GeoJSON 实体/属性/关系/空间成果导出
- 用户管理、全流程任务监控、AI 调用时间/模型/功能/状态/异常审计
- DeepSeek/Qwen API 地址、模型名称、Key、temperature、Prompt 模板配置及运行时应用
- 三份地质 Demo 文本与批量导入脚本
- 独立部署文档、完整接口文档、测试报告和三张系统截图
- V008 系统管理与模型配置数据库迁移

### 修改

- 管理接口限制为 ADMIN 角色
- 三端版本与 Dashboard 更新为 v1.0.0 最终交付状态
- README 汇总 Phase 1–8 完整能力和交付入口

## [0.7.0] - 2026-07-20

### 新增

- Phase 7 Neo4j 地质知识图谱同步，覆盖六类节点与四类核心关系
- 节点查询、1–3 层关系展开和六跳以内最短路径查询
- ECharts Graph 力导向图、节点证据卡和路径查询工作台
- BAAI/bge-m3 延迟加载 Embedding 与 Qdrant 文档块向量索引
- Qdrant 检索、Neo4j 上下文和 DeepSeek/Qwen 联合的证据约束问答
- 回答关联实体、空间坐标、来源资料、页码、段落和相似度展示
- 图谱/RAG 契约测试、后端同步编排测试及桌面/窄屏浏览器验收

### 修改

- 上游解析、识别、知识抽取或空间化重跑时使旧图谱状态失效
- ECharts 改为按需组件加载，控制图谱页面产物体积
- Dashboard、侧栏和三端版本更新为 Phase 7 / 0.7.0

## [0.6.0] - 2026-07-20

### 新增

- Phase 6 地名、坐标、矿点、钻孔、断裂和调查区域空间信息抽取
- Point、LineString、Polygon GeoJSON 严格校验与可选地名解析
- `spatial_object` PostGIS 空间表、GIST 索引及 V006 数据库迁移脚本
- Spring Boot 异步空间化编排、任务状态和空间对象查询接口
- OpenLayers 标准/卫星底图、图层显隐、缩放、图例、比例尺和坐标显示
- 距离/面积量算、地图对象详情和地图—原文证据双向联动
- 空间抽取契约测试、后端编排测试与桌面/窄屏浏览器验收

### 修改

- 文档解析、实体识别或知识抽取重跑时自动使旧空间化结果失效
- Dashboard、侧栏和三端版本更新为 Phase 6 / 0.6.0

## [0.5.0] - 2026-07-20

- 标准名、别名和未匹配三类术语标准化结果
- 知识任务入口、关系证据、属性卡片和术语对照核查页面
- 属性关系 AI 契约测试、后端编排测试和词典匹配测试

### 修改

- Dashboard、侧栏和三端版本更新为 Phase 5 / 0.5.0

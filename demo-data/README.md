# 优化演示数据包

本目录提供四份相互关联、格式不同的演示资料，用于验证“资料入库、解析/OCR、实体识别、属性关系、术语标准化、文本空间化、知识图谱、智能问答和成果导出”完整流程。所有内容均为虚构的功能演示样本，不代表真实勘查成果。

| 序号 | 文件 | 格式与验证重点 |
| --- | --- | --- |
| 01 | `铜绿山矿段综合地质调查报告` | UTF-8 TXT；主流程资料，覆盖五类属性、六类关系及真实坐标点线面 |
| 02 | `ZK001钻孔地质编录表` | DOCX；验证 Word 在线预览、章节解析、钻孔和矿体数值 |
| 03 | `铜绿山矿段矿化特征简报` | 可检索 PDF；验证 PDF 解析、段落证据和矿化术语 |
| 04 | `F1断裂野外记录扫描件` | 图像型 PDF；验证 PaddleOCR、坐标、倾向和倾角识别 |

推荐现场先在“系统管理”点击“恢复标准演示数据”，获得确定性的完整结果；随后上传 02、03、04，分别展示 Word 预览、PDF 解析和扫描 PDF OCR。批量上传可运行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\scripts\import-demo-data.ps1
```

详细操作见 `docs/系统演示操作指南.docx`，逐字讲解稿见 `docs/项目功能演示讲稿.docx`。

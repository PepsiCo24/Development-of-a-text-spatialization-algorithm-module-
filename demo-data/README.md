# 地质演示数据

本目录提供三份 UTF-8 文本资料，用于完整演示“上传 → 解析 → 实体/关系 → 空间化 → 图谱 → 问答 → 导出”流程。

| 文件 | 区域 | 主题 |
| --- | --- | --- |
| `01-鄂东南铁铜矿调查.txt` | 湖北鄂东南 | 花岗岩、断裂、矽卡岩型铁铜矿化 |
| `02-宜昌磷矿地层调查.txt` | 湖北宜昌 | 震旦系—寒武系地层与磷矿层 |
| `03-大冶矿区钻孔记录.txt` | 湖北大冶 | 钻孔、矿体厚度、品位和坐标 |

可在“资料资源池”逐份上传，或启动系统后运行 `scripts/import-demo-data.ps1` 批量导入。演示资料为项目内构造的测试样本，不代表正式勘查结论。

完整的现场演示步骤、推荐话术、问答示例和成果导出说明见 [`docs/demo-tutorial.md`](../docs/demo-tutorial.md)。
